# Ensolvers todo-server

Backend REST API Server to Create, Edit and Delete Tasks managed into folders.

Built with express generator.

Routes Availables: 
  '/tasks',
  '/tasks/:id',
  '/folders',
  '/folders/:id'

## Technologies and libraries used:
- Node.js v12.19.0
- cors: ^2.8.5,
- dotenv: ^10.0.0,
- express: ~4.16.1,
- promise-mysql: ^5.0.3
- MySQL: 5.7
